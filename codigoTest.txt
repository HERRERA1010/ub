        <h3>¿Cómo hacer transiciones suaves con CSS?</h3>
        <p>Para evitar efectos bruscos, usa <code>transition</code> o <code>@keyframes</code>:</p>
        
        <ul>
            <li><strong>Method 1 - Transition simple:</strong><br>
                <code>.elemento { transition: transform 0.3s ease; }</code><br>
                <code>.elemento:hover { transform: scale(1.3); }</code>
            </li>
            
            <li><strong>Method 2 - Keyframes personalizados:</strong><br>
                <code>@keyframes smoothScale {<br>
                &nbsp;&nbsp;0% { transform: scale(1); }<br>
                &nbsp;&nbsp;100% { transform: scale(1.3); }<br>
                }</code><br>
                <code>.elemento:hover { animation: smoothScale 0.4s ease forwards; }</code>
            </li>
            
            <li><strong>Timing functions útiles:</strong><br>
                <code>ease</code>, <code>ease-in-out</code>, <code>cubic-bezier(0.68,-0.55,0.265,1.55)</code>
            </li>
            
            <li><strong>Propiedades que puedes animar:</strong><br>
                <code>transform</code>, <code>color</code>, <code>opacity</code>, <code>letter-spacing</code>, etc.
            </li>
        </ul>
